# MVP 요구사항

원본 사양서의 기능 범위와 완료 기준을 구현 가능한 요구사항 및 추적 항목으로 분리하는 문서입니다.

초기 기준 문서: `Hotel_CallCenter_Chat_MVP_Design.md`

## 2026-07-23 확정 요구사항

- Guest는 `WAITING`에서도 메시지를 즉시 DB에 저장하며 대기 시간만으로 만료하지 않는다.
- 대화를 여는 동작은 담당 Agent만 배정하며, 15분은 담당 Agent의 첫 답변 메시지가 DB에 저장된 시점부터 시작한다.
- 호텔별 일본어·영어 첫 자동 안내문을 신규 상담의 SYSTEM 메시지로 저장한다.
- Agent 화면은 LINE형 `Current chat room / Log`, 검색·호텔·언어 필터와 한국어·일본어 UI를 제공한다.
- 관리자 화면은 호텔·룸 관리, 호텔별 자동 안내문, Agent 관리 순서의 대시보드를 사용한다.
